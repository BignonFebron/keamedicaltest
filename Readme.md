# KeaMedicaltest Application

Suivre les steps suivantes pour lancer et tester le projet

## Steps to Setup

1. Installer les dependences

```bash 
npm install
```

2. Demarrer le Server

```bash
node server.js
```

les api seront disponible sur le lien <http://localhost:3000>
